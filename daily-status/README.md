# DailyStatus (HTMX Demo)

A corporate timesheet and status reporting tool.

## Features
-   **High-Volume Search**: Instantly filters 50+ history items server-side.
-   **Form Handling**: Submits complex forms and updates the list without page reloads.
-   **UI**: Clean "Corporate Light" theme.

## Installation

1.  Navigate to the directory:
    ```bash
    cd daily-status
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the server:
    ```bash
    node server.js
    ```

4.  Open your browser at **[http://localhost:3001](http://localhost:3001)**.
