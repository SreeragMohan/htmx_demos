# TaskFlow (HTMX Demo)

A premium task management dashboard demonstrating reactive UI patterns with HTMX.

## Features
-   **Active Search**: Real-time filtering as you type.
-   **Inline Editing**: Click-to-edit task descriptions.
-   **Out-of-Band Swaps**: Automatically updates "Stats" counters when tasks change.
-   **UI**: Dark mode with Glassmorphism effects.

## Installation

1.  Navigate to the directory:
    ```bash
    cd taskflow
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Start the server:
    ```bash
    node server.js
    ```

4.  Open your browser at **[http://localhost:3000](http://localhost:3000)**.
