# Premium HTMX Real-World Demos

A collection of high-quality, "real-world" web applications demonstrating the power of HTMX for modern web development. These demos showcase how to build premium, reactive user interfaces without the complexity of heavy client-side frameworks like React or Angular.

**Author:** Sreerag Mohan

## Demos Included

1.  **[TaskFlow](./taskflow)** - A collaborative task management dashboard (Dark Mode, Glassmorphism).
2.  **[DailyStatus](./daily-status)** - A corporate timesheet and status reporting tool (Corporate Light Theme).

## Prerequisites

-   [Node.js](https://nodejs.org/) (v14 or higher)
-   npm (comes with Node.js)

## Getting Started

Clone the repository and navigate to the desired demo folder.

### Running TaskFlow
```bash
cd taskflow
npm install
node server.js
```
Visit: http://localhost:3000

### Running DailyStatus
```bash
cd daily-status
npm install
node server.js
```
Visit: http://localhost:3001
